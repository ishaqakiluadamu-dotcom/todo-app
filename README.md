# To-Do Web App

A full-stack To-Do Web Application built entirely using Python (Flask), SQLite, and vanilla JavaScript.  
Designed and developed on a mobile environment (Termux) to showcase backend, frontend, and database integration.

## Features
- Add, delete, update, and view tasks
- Persistent storage using SQLite
- RESTful API (GET / POST / PUT / DELETE)
- Interactive web interface
- Professional full-stack project workflow

## Tech Stack
- Python 3.12
- Flask
- SQLite
- HTML / JavaScript / CSS

## Usage
1. Clone repository  
2. Install dependencies: `pip install flask flask-cors`  
3. Run server: `python app.py`  
4. Open browser: `http://127.0.0.1:5000/`
